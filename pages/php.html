Durante mis cursos en PHP , fueron orientados al uso de las base de datos SQL y la extraccion/inclusion de los mismos en formato JSON. Estos fueron todos bastantes simples acciones CRUD y fueron combinadas con Angular(Interfaz de usuario) y Nodejs (Servidor continuo) .
Estas simplemente realizaban operaciones sobre la base de datos llamadas por el interfaz correspondiente
En angular fueron usadas como una simple herramienta de gestion empresarial, consulta de articulos ,realizacion de pedido, cambios de precio, alertas de stock etc.
En Nodejs para aprovechar la falta de funcionalidades en comparacion de otras base de datos se uso un bucle para emular un juego web service de gestion de recursos que actualizaba el crecimiento de estos cada segundo a traves de la ejecucion de ordenes PHP
